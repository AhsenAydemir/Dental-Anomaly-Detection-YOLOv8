# Dental-Anomaly-Detection-YOLOv8
Bu proje, panoramik dental röntgen görüntülerinde anomalileri (çürükler, dolgular, eksik dişler vb.) otomatik olarak tespit etmek için geliştirilmektedir.

## Proje Öne Çıkanlar
* **Veri Seti:** 13.000'den fazla etiketlenmiş dental görüntü.
* **Model:** YOLOv8 (Ultralytics) mimarisi.
* **Arayüz:** Streamlit ile web tabanlı dashboard.

## Dosyalar
* `app.py`: Streamlit arayüz kodları.
* `best.pt`: Eğitilmiş model ağırlıkları.
